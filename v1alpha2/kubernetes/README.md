# Kubernetes OPM resources

## Summary

Will contain Kubernetes-native resource types so that modules can declare platform-specific Kubernetes
resources directly.

## Status

**Deferred.** This module is not in scope for the current work cycle. Gateway API resources are handled
by the dedicated `gateway_api/` extension module. cert-manager resources are handled by the `cert_manager/`
extension module. Work on this module will begin in a future iteration once those two modules are complete.
